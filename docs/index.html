Ignition Browser

Disposable, isolated browser for macOS on Apple Silicon. Every link opens in a throwaway Firefox microVM — fresh each time, nothing persists.

Download

Shortcuts

⌃⌥RReset — fresh session
⌃⌥SSnapshot VM
⌃⌥XClose session

Host keys use ⌃⌥ (Ctrl+Alt); standard Firefox shortcuts (address bar, reload, find, zoom…) all go to Firefox.

What it is

  • Isolated — pages run in a hardware VM, not on your Mac.
  • Disposable — in-RAM disk; every session is a clean snapshot.
  • User-mode net — NAT only; host + LAN blocked.
